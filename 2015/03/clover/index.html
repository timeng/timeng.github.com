Clover-你的Windows资源管理器从未如此高效

发表于 2015-03-04   |   分类于 software   |  

软件推荐第一期，推荐一款效率工具–Clover。

Clover是Windows Explorer资源管理器的一个扩展，作者是国人，其从2012年发布，我算是最早一批用户，目前Clover版本已经迭代至3.X，至今Clover一直是我的必装软件之一。

主要功能特性

方便的 Tab 页功能
要掌握功能强大，操作简单的标签页，只需记住Ctrl+T新开页面，Ctrl+W关闭页面，Ctrl+Tab切换页面，工作效率提高何止一倍！

操作系统无缝集成
Clover 通过插件的形式集成到 Windows Explorer，保留您通常的使用习惯，无需学习新的文件管理操作，马上就可以使用啦。

快如闪电的书签栏
按Ctrl+D添加当前路径，或者直接将文件夹拖入书签栏。再也不用到处寻找要访问的文件夹了，瞬间到达，何等痛快！

更多说明，请访问Clover官网 http://cn.ejie.me/
