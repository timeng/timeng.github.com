3. 需要注意的地方

  1. 使用内建函数，性能要好于lambda，所以不要滥用lambda
  2. 对于只调用一次的函数，写成lambda更简单，因为无需使用def定义一个新的函数，lambda调用结束就会被销毁
  3. lambda已经隐含了return，它是一个表达式，而不是语句
  4. lambda只能写单行表达式
