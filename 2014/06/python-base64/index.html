Python2.x VS 3.x 之base64

发表于 2014-06-02   |   分类于 python   |  

1. python2和python3进行base64的相同点

都引入base64模块，编码调用b64encode()函数，解码调用b64decode()函数

import base64
base64.b64encode()
base64.b64decode()

2. 异同点

2.1 python2.x

在python2中，b64encode()函数的入参是string类型，返回值是string类型

输出结果: 5ZOI5ZOI

base64解码类似，入参是string类型，返回值也是string
b64decode(b64)

2.2 python3.x

在python3中，b64encode()函数的入参是byte类型，返回值是byte类型。

调用时，必须先将字符串encode成byte类型，相反的，对函数返回值进行decode，得到base64字符串。以下实例

输出结果: 5ZOI5ZOI

base64解码类似，入参是byte类型，返回值也是byte类型，decode()之后得到字符串
